# playwright-codepipeline-demo
A demonstration project showing how a modern CI/CD pipeline can be built using Playwright, Python Flask, GitHub and AWS CodePipeline.

The project includes a simple mock web application based on a postcode lookup service, together with an automated Playwright test suite used to validate the application's behaviour.

The purpose of the repository is educational, demonstrating how a business requirement can be transformed into a BDD scenario, automated using Playwright, placed under source control, and ultimately integrated into a cloud-based CI/CD workflow.

Technologies used:

* Python Flask
* Playwright
* TypeScript
* GitHub
* AWS CodePipeline
* AWS CodeBuild

This repository forms part of the "CI/CD Pipelines Explained for Testers" video series.
